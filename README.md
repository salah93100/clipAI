## Aperçu

**ClipAI** est une application web qui permet aux créateurs de contenu de monter des vidéos de manière efficace et rapide. Grâce à l'intégration de composants UI modernes et de bibliothèques puissantes, ClipAI offre une expérience utilisateur fluide et intuitive.

## Installation

Pour installer et exécuter ce projet localement, suivez ces étapes :

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/salah93100/clipAI.git
   ```

2. Accédez au répertoire du projet :
   ```bash
   cd clipAI
   ```

3. Installez les dépendances :
   ```bash
   npm install
   ```

## Utilisation

Pour démarrer le serveur de développement, exécutez :

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) pour voir votre application dans le navigateur.

## Fonctionnalités

- **Montage Vidéo Rapide** : Importez, éditez et exportez des vidéos en quelques clics.
- **Interface Intuitive** : Utilisez une interface utilisateur moderne et réactive.
- **Outils de Création** : Accédez à une variété d'outils pour améliorer vos vidéos, y compris des filtres, des transitions, et plus encore.
- **Support Multi-Format** : Travaillez avec différents formats vidéo pour une flexibilité maximale.

## Scripts Disponibles

- `npm run dev` : Démarre le serveur de développement.
- `npm run build` : Compile l'application pour la production.
- `npm run start` : Démarre le serveur de production.
- `npm run lint` : Lint le code pour les erreurs.

## Dépendances

Le projet utilise les bibliothèques suivantes :

- **Next.js** : Framework React pour le développement d'applications web.
- **React** : Bibliothèque JavaScript pour construire des interfaces utilisateur.
- **Radix UI** : Composants d'interface utilisateur accessibles et personnalisables.
- **Tailwind CSS** : Framework CSS pour un design rapide et réactif.

Pour une liste complète des dépendances, consultez le fichier `package.json`.
